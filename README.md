Project Overview

The Titanic Data Storytelling App is an interactive web application developed using Python and Streamlit. The application analyzes the Titanic dataset and presents meaningful insights through exploratory data analysis (EDA), visualizations, and data-driven storytelling.

The purpose of this project is to understand the factors that influenced passenger survival and communicate findings in a simple and visually engaging manner.

Dataset Introduction

The Titanic dataset contains information about passengers who traveled aboard the RMS Titanic. The dataset includes details such as:

Survival Status
Passenger Class
Gender
Age
Fare
Embarkation Port
Family Information

This dataset is widely used for learning data analysis, visualization, and machine learning concepts.

Exploratory Data Analysis (EDA)

The application performs exploratory data analysis to understand the structure and characteristics of the dataset.

Data Cleaning and Preparation
Visualizations

Insights and Findings
Key Findings
Female passengers had a higher survival rate compared to male passengers.
First-class passengers had better chances of survival than passengers in lower classes.
Younger passengers generally had higher survival rates.
Ticket fare showed a relationship with survival probability.
Most passengers embarked from Southampton.
Final Conclusion and Recommendations
Conclusion

The analysis demonstrates that factors such as gender, passenger class, age, and fare significantly influenced survival outcomes during the Titanic disaster.
